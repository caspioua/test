Referer Test Page

https://textslashplain.com/2019/10/16/privacy-tweaks-limiting-referrer/

referrerpolicy=unsafe-url(http)
referrerpolicy=unsafe-url(https)
Send the full URL (lacking only auth info and fragment), even on navigations from HTTPS to HTTP
referrerpolicy=no-referrer
no-referrer – Do not send a Referer
referrerpolicy=no-referrer-when-downgrade(http)
referrerpolicy=no-referrer-when-downgrade(https)
no-referrer-when-downgrade – Don’t send the Referer when navigating from HTTPS to HTTP. [The longstanding default behavior of browsers]
referrerpolicy=strict-origin-when-cross-origin(http)
referrerpolicy=strict-origin-when-cross-origin(https)
strict-origin-when-cross-origin – For a same-origin navigation, send the URL. For a cross-origin navigation, send only the Origin of the referring page. Send nothing when navigating from HTTPS to HTTP. [The new default]
referrerpolicy=origin-when-cross-origin(http)
referrerpolicy=origin-when-cross-origin(https)
origin-when-cross-origin – For a same-origin navigation, send the URL. For a cross-origin navigation, send only the Origin of the referring page. Send the Referer even when navigating from HTTPS to HTTP
referrerpolicy=same-origin
same-origin – Send the Referer only for same-origin navigations
referrerpolicy=origin
origin – Send only the Origin of the referring page
referrerpolicy=strict-origin
strict-origin – Send only the Origin of the referring page; send nothing when navigating from HTTPS to HTTP
